# Package index

## Compute partial dependence

Compute partial dependence functions and individual conditional
expectation (ICE) curves from various fitted model objects.

- [`partial()`](https://bgreenwell.github.io/pdp/reference/partial.md) :
  Partial Dependence Functions
- [`exemplar()`](https://bgreenwell.github.io/pdp/reference/exemplar.md)
  : Exemplar observation

## Plotting

Display partial dependence functions and ICE curves using lightweight
base R graphics (via tinyplot) or lattice graphics.

- [`plot(`*`<partial>`*`)`](https://bgreenwell.github.io/pdp/reference/plot.partial.md)
  [`plot(`*`<ice>`*`)`](https://bgreenwell.github.io/pdp/reference/plot.partial.md)
  [`plot(`*`<cice>`*`)`](https://bgreenwell.github.io/pdp/reference/plot.partial.md)
  : Plotting Partial Dependence Functions
- [`plotPartial()`](https://bgreenwell.github.io/pdp/reference/plotPartial.md)
  : Plotting Partial Dependence Functions (deprecated)
